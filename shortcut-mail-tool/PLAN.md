# 簡易ショートカットメール支援システム 検討メモ

## 目的
1つのショートカットコマンドでExcelまたは軽量ツールを起動し、定型のメールアドレスや会議案内文を1クリックでコピーする環境を提供する。

## 最小構成案
- Windowsのバッチファイルを作成し、単一キー（例: Ctrl+Alt+M）のショートカットに割り当てる。
- バッチファイルはローカルに保存したExcelファイルを起動する。
- Excelファイルにはメールアドレス一覧と会議案内文面をシート分けで用意する。
- 各セルにコピー用VBAボタンを配置し、クリックでクリップボードにコピー。

## Excelより軽量な案
- CSVファイルを用意し、PowerShellスクリプトでメニューを表示し選択した項目を`clip`コマンドでクリップボードへ。
- HTML Application (HTA) を利用し、HTMLとJavaScriptで軽いGUIを構築。Windowsに標準搭載されている`mshta.exe`を用いるため追加インストール不要。

## ブラッシュアップのアイデア
- Microsoft OutlookのQuick Partsを活用すれば、Outlook上で定型文と連絡先管理が可能で、外部アプリを開かずに済む。
- Excelマクロでは`Application.OnKey`を利用してショートカットキーを割り当て、Excel起動後に自動実行する仕組みを導入する。
- PowerShellスクリプトに`Out-GridView`を組み合わせ、GUI風のリストから選択してコピーする方法もある。
- Windowsのタスクスケジューラやスタートアップ登録で常駐させ、更に素早い操作が可能。

## 検討課題
- クリップボード操作時の権限問題やセキュリティポリシーを確認する。
- 長期運用を想定した場合、リストの更新を簡便にするためデータをCSVやJSONで分離するなど管理方法を検討する。
